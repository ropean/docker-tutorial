## 快速开始

立即运行你的第一个 Docker 容器：

```bash
# 拉取并运行 hello-world 镜像
docker run hello-world
```

这个简单的命令会：
1. 检查本地是否有 hello-world 镜像
2. 如果没有，从 Docker Hub 下载
3. 创建并启动一个新容器
4. 在容器中执行程序并输出消息
5. 容器执行完毕后自动停止
